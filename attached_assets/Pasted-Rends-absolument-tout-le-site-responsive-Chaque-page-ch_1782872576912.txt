ÉTAPE 3 — SYSTÈME DE BREAKPOINTS UNIQUE

Ces breakpoints s'appliquent partout, sans exception :

Mobile (défaut, 0–639px)
  Layout en une seule colonne. Rien ne dépasse horizontalement.
  Tout texte a du padding. Boutons full-width si isolés.

sm: 640px
  Ajustements mineurs. Grilles passent à 2 colonnes max.

md: 768px
  Layouts à 2 colonnes. Sidebar peut apparaître en overlay.

lg: 1024px
  Layout desktop complet. Sidebar fixe visible. Grilles 3+.

xl: 1280px
  Affinages de densité uniquement.

Règles fixes par type de composant :

Grilles de cards :
  grid-cols-1 sm:grid-cols-2 lg:grid-cols-3
  Jamais grid-cols-3 sans variantes mobile.

Texte display/heading :
  text-display-hero-mobile md:text-display-hero
  Tout titre ≥ 22px doit avoir une variante mobile.

Navigation :
  hidden lg:flex  → liens desktop
  flex lg:hidden  → burger mobile
  Jamais de liens compressés en minuscule sur mobile.

Sidebar dashboard :
  hidden lg:flex lg:w-[224px]  → sidebar desktop
  Sous lg: drawer ou bottom-nav, jamais fixe par-dessus le contenu.

Formulaires :
  Tous les inputs en w-full.
  Champs côte à côte : flex-col sm:flex-row.

Images :
  w-full h-full object-cover toujours.
  Jamais de width fixe en px sur une image de layout.

Modals/Dialogs :
  w-full mx-4 sur mobile, max-w-md mx-auto sur md:
  Padding interne minimum p-6 à toutes les tailles.

Boutons :
  Hauteur minimum 44px sur mobile (touch target).
  Jamais tronqués ou coupés.
